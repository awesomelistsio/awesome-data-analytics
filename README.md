## Foundations & Concepts

- [Analytics Overview](https://en.wikipedia.org/wiki/Analytics) – High-level overview of analytical methods and use cases.
- [CRISP-DM](https://www.ibm.com/topics/crisp-dm) – Widely used framework for analytics and data mining projects.
- [DIKW Pyramid](https://en.wikipedia.org/wiki/DIKW_pyramid) – Conceptual model linking data, information, knowledge, and wisdom.
- [Descriptive vs Predictive Analytics](https://www.tableau.com/learn/articles/descriptive-analytics) – Comparison of core analytics types.
- [Analytics Maturity Models](https://www.gartner.com/en/information-technology/glossary/analytics-maturity-model) – Frameworks for assessing analytics capability.

## Analytics Platforms & BI Tools

- [Tableau](https://www.tableau.com/) – Industry-leading business intelligence and visualization platform.
- [Power BI](https://powerbi.microsoft.com/) – Microsoft analytics and dashboarding platform.
- [Looker](https://cloud.google.com/looker) – Semantic-model-driven BI and analytics platform.
- [Qlik Sense](https://www.qlik.com/us/products/qlik-sense) – Associative analytics and BI platform.
- [Apache Superset](https://superset.apache.org/) – Open-source data exploration and visualization platform.
- [Metabase](https://www.metabase.com/) – Open-source analytics and dashboards for teams.

## Data Warehousing & OLAP

- [Snowflake](https://www.snowflake.com/) – Cloud-native data warehouse for analytics workloads.
- [Google BigQuery](https://cloud.google.com/bigquery) – Serverless analytics data warehouse.
- [Amazon Redshift](https://aws.amazon.com/redshift/) – Managed data warehouse on AWS.
- [ClickHouse](https://clickhouse.com/) – High-performance columnar database for analytics.
- [Apache Druid](https://druid.apache.org/) – Real-time OLAP database for event analytics.
- [Apache Pinot](https://pinot.apache.org/) – Distributed OLAP datastore for low-latency analytics.

## SQL & Query Engines

- [PostgreSQL](https://www.postgresql.org/) – Advanced relational database widely used for analytics.
- [DuckDB](https://duckdb.org/) – In-process analytical SQL engine optimized for OLAP.
- [Trino](https://trino.io/) – Distributed SQL query engine for large datasets.
- [Presto](https://prestodb.io/) – High-performance distributed SQL query engine.
- [Apache Spark SQL](https://spark.apache.org/sql/) – SQL analytics engine built on Apache Spark.
- [SQLite](https://www.sqlite.org/) – Lightweight SQL database commonly used for local analytics.

## Data Transformation & Modeling

- [dbt](https://www.getdbt.com/) – Analytics engineering tool for transforming data in warehouses.
- [Apache Airflow](https://airflow.apache.org/) – Workflow orchestration platform for analytics pipelines.
- [Dagster](https://dagster.io/) – Data orchestration platform focused on observability and testing.
- [Meltano](https://meltano.com/) – Open-source ELT platform for analytics pipelines.
- [Apache NiFi](https://nifi.apache.org/) – Visual tool for data flow and transformation.
- [SQLMesh](https://sqlmesh.com/) – Versioned and testable SQL-based data transformations.

## Product & Behavioral Analytics

- [Amplitude](https://amplitude.com/) – Product analytics platform for user behavior analysis.
- [Mixpanel](https://mixpanel.com/) – Event-based analytics for product teams.
- [PostHog](https://posthog.com/) – Open-source product analytics platform.
- [Heap](https://heap.io/) – Automatic event capture for behavioral analytics.
- [Segment](https://segment.com/) – Customer data platform for analytics integration.

## Web & Marketing Analytics

- [Google Analytics](https://analytics.google.com/) – Web traffic and user behavior analytics platform.
- [Google Tag Manager](https://tagmanager.google.com/) – Tag management system for analytics tracking.
- [Adobe Analytics](https://business.adobe.com/products/analytics/adobe-analytics.html) – Enterprise web analytics solution.
- [Matomo](https://matomo.org/) – Privacy-focused open-source web analytics platform.
- [Hotjar](https://www.hotjar.com/) – User behavior analytics via heatmaps and recordings.

## Visualization Libraries

- [Matplotlib](https://matplotlib.org/) – Foundational plotting library for Python analytics.
- [Seaborn](https://seaborn.pydata.org/) – Statistical visualization library for Python.
- [Plotly](https://plotly.com/) – Interactive charts and dashboards.
- [Altair](https://altair-viz.github.io/) – Declarative visualization library for analytics.
- [D3.js](https://d3js.org/) – JavaScript library for custom data visualizations.
- [Apache ECharts](https://echarts.apache.org/) – Powerful visualization library for analytics dashboards.

## Streaming & Real-Time Analytics

- [Apache Kafka](https://kafka.apache.org/) – Distributed event streaming platform.
- [Apache Flink](https://flink.apache.org/) – Stream processing framework for real-time analytics.
- [Apache Spark Streaming](https://spark.apache.org/streaming/) – Real-time data processing on Spark.
- [ksqlDB](https://ksqldb.io/) – Streaming SQL engine for Kafka data.
- [Materialize](https://materialize.com/) – Streaming database for real-time analytics.
## Analytics Engineering

- [Analytics Engineering](https://www.getdbt.com/what-is-analytics-engineering/) – Emerging discipline combining analytics and engineering.
- [Semantic Layers](https://www.looker.com/platform/semantic-layer) – Abstraction layers for consistent analytics metrics.
- [Metric Stores](https://transform.co/) – Centralized systems for managing analytics metrics.
- [Cube](https://cube.dev/) – Headless BI platform for analytics APIs.
- [Lightdash](https://www.lightdash.com/) – Open-source BI tool built on dbt.

## Metrics, KPIs & Experimentation

- [North Star Metrics](https://amplitude.com/blog/north-star-metric) – Core metrics guiding product growth.
- [A/B Testing](https://www.optimizely.com/optimization-glossary/ab-testing/) – Experimentation framework for decision-making.
- [Optimizely](https://www.optimizely.com/) – Experimentation and feature testing platform.
- [Statsig](https://www.statsig.com/) – Product analytics and experimentation platform.
- [Eppo](https://www.geteppo.com/) – Experimentation and analytics platform for data teams.

## Governance, Quality & Observability
